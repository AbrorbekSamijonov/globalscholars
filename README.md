## Global Scholars Prep Landing Page

This is a focused, high-conversion landing page for the **Global Scholars Prep** program.

- **Tech stack**: plain HTML, CSS, and a tiny bit of vanilla JavaScript.
- **Entry point**: open `index.html` in a browser.
- **Branding**:
  - Primary color: Oxford Navy `#002147`
  - Accent color: Academic Gold `#C5A059` (used sparingly for key words like “Win”, “Strategy”, and “Finalist”)
  - Background: Paper White `#FDFDFD`
  - Fonts: `Cinzel` for headlines (diploma / elite feel), `Inter` for body text.

### Customization

- Replace the mentor photos in the hero card:
  - `images/mentor-1.jpg`
  - `images/mentor-2.jpg`
- Update the links in:
  - The main CTA buttons (`#apply` section).
  - The Telegram handle in the footer.
